# Gerador de Senhas Seguras – MVP Acadêmico

## Objetivo
Criar um gerador de senhas seguras executado via linha de comando (CLI),
desenvolvido com auxílio do GitHub Copilot, para fins acadêmicos.

## Funcionalidades
- Gerar senhas aleatórias
- Definir o tamanho da senha
- Escolher se a senha deve conter:
- Letras maiúsculas
- Letras minúsculas
- Números
- Caracteres especiais
- Validar as entradas do usuário
- Exibir a senha gerada no terminal

## Requisitos Técnicos
- Linguagem: Python
- Utilizar apenas bibliotecas padrão do Python
- Interface via terminal (CLI)
- Código simples, legível e comentado
- Testes unitários básicos com cenários de erro
- README detalhado explicando:
- Como executar o projeto
- Como o GitHub Copilot (IA generativa) foi utilizado
